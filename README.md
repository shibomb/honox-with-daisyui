# HonoX with daisyUI

This is the simple template of HonoX with daisyUI v5.
- HonoX with tailwind CSS v4
- daisyUI v5
- Includes a Theme Selector inspired by the daisyUI website
- Includes a tailwind Typography demo page from daisyUI docs
